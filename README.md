# FUTURE_DS_02

## 📊 Task Overview
**Task:** Customer Retention & Churn Analysis  
**Objective:** Analyze customer data to identify churn patterns, key retention drivers, and customer lifetime trends for a subscription-based business.  
**Track Code:** DS  
**Deliverable:** Retention analysis dashboard/report highlighting churn reasons, retention trends, and actionable recommendations.

---

## 🛠 Tools & Technologies
- **Excel** → Imported Kaggle dataset and performed data cleaning/preprocessing  
- **Power BI** → Analysis, cohort metrics, and interactive dashboard creation  

---

## 🔄 Workflow
1. **Data Import:** Downloaded customer dataset from Kaggle and loaded into Excel.  
2. **Preprocessing (Excel):**  
   - Cleaned missing values and standardized fields  
   - Prepared structured dataset for analysis  
3. **Dashboard Development (Power BI):**  
   - Built retention and churn KPIs  
   - Designed cohort analysis visuals  
   - Created charts for customer lifetime value and retention trends  

---

## 📈 Key Insights
- Identified churn patterns across customer cohorts.  
- Highlighted key drivers of retention (e.g., subscription length, engagement).  
- Analyzed customer lifetime metrics to detect high-value segments.  
- Generated actionable recommendations to reduce customer loss.
